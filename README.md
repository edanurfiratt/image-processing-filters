# Image Processing Filters Project

This project demonstrates fundamental image processing techniques using Python.

## 📌 Implemented Filters
- Average (Mean) Filter
- Gaussian Filter
- Median Filter
- Sobel Edge Detection
- Laplacian Filter

## ⚙️ Technologies
- Python
- NumPy
- Matplotlib
- Scikit-image
- SciPy

## 🎯 Purpose
This project aims to analyze how different filters affect an image in terms of:
- Noise reduction
- Image smoothing
- Edge detection
